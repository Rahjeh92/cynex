# 🌐 CYNEX Website

A responsive landing page for **CYNEX**, a company that provides **Networking Installation, Security Solutions, and Website Development**.  
This project is fully bilingual (**English / Arabic**) with an interactive language switcher.

---

## 🚀 Features
- Responsive design (works on Desktop, Tablet, Mobile).
- Bilingual content (English / Arabic) with toggle button.
- Smooth scroll navigation.
- Contact form integrated with [Formspree](https://formspree.io).
- Clean HTML, CSS, and JavaScript structure.
- RTL (Right-to-Left) support when switching to Arabic.
- Modern hover effects (buttons glow yellow, cards highlight on hover).

---

## 🛠️ Technologies
- **HTML5**
- **CSS3** (Responsive + Flex/Grid)
- **JavaScript (Vanilla)**  
- **Formspree** (for contact form handling)
